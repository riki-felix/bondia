# Rutas principales Bondia

## Global

- `/` — Dashboard
- `/cartera` — Cartera virtual
- `/inversiones` — Tabla inversiones
- `/liquidaciones` — Liquidaciones
- `/propiedades`, `/propiedades/nuevo`, `/propiedades/[id]`
- `/ajustes` — Métodos de pago, feature tasks

## Casa

- `/casa/control`
- `/casa/gastos`, `/casa/ingresos`
- `/casa/activos`, `/casa/activos/nuevo`, `/casa/activos/[id]`
- `/casa/categorias`

## Sanyus

Mismas rutas con prefijo `/sanyus/` (ver `SANYUS_CONFIG.routes` en `bloqueConfig.ts`).
